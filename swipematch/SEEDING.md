# SEEDING.md — Solve supply before demand

The single biggest risk to this app is the **cold-start / liquidity trap**: a new user
opens the Room, sees an empty or irrelevant deck, and never comes back. Two-sided
networks like this one mostly die in *year 2* for exactly this reason. Engagement
mechanics (streaks, daily deck, Growth score) are a **multiplier on liquidity, never a
substitute** for it. So before any public launch, we manually create density.

This is a go-to-market action, not an app feature. **No production data is inserted by
the codebase** — every seed profile below is a real, recruited person.

## Rule 1 — Pick ONE scene first
Density beats breadth. Uber recruited drivers city-by-city; we do the same with people.

Choose a single tight community where the members already overlap in real life, e.g.:
- one university's student-founder / entrepreneurship society, **or**
- one city's early-stage startup + creator scene.

Win ~100 real, active profiles in that ONE scene before opening a second. A focused 500
who all know each other beats a diffuse 50,000 strangers.

## Rule 2 — Recruit ~100 real profiles BEFORE public launch
Hand-recruit. DM, in-person at meetups, campus clubs. Onboard them yourself if needed.
Target: ~100 complete profiles **with intro videos** live before you let cold installs in.

## Rule 3 — Every seed profile must be "good" (or it ranks poorly and reels look empty)
The Room ranks by intent fit → region → profile completion, and the Reels front door
shows intro videos first. A thin profile sinks to the bottom and breaks the first
impression. A good seed profile fills **all** of:

- **persona** — Student / Founder / Creator / Professional / Investor / Researcher
- **≥ 1 connection_intent** — what they want (mentorship, collaborators, investors,
  hiring, friends, knowledge). This is the #1 ranking signal — never leave it empty.
- **city + country** — drives the regional tiebreak; keep the first scene in one city.
- **working_toward** — their current goal, one sentence.
- **intro video** — a real ≤60s clip. This is the wedge and the Reels front door; a
  profile without one falls back to posts/achievements, so seed at least 1–2 posts too
  if no video yet.

## Why this order matters
- Intent-primary matching (migration 020) only pays off if people actually set intents.
- Reels-first front door only feels good if enough seeds have videos.
- Same-scene seeding makes the region tiebreak cluster real, connectable people at the top.

Get these 100 right and the retention mechanics finally have something to amplify.
